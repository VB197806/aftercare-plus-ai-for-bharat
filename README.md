AfterCare+

AfterCare+ is an AI-powered post-discharge continuity intelligence platform built on AWS. It converts discharge summaries into structured care workflows, drives medication adherence, enables follow-up booking, and triggers rule-based escalation workflows.
Built for AWS AI for Bharat Hackathon.

Architecture Highlights:
a. Serverless AWS architecture
b. Amazon Textract for document extraction
c. Amazon Bedrock for structured care plan generation
d. DynamoDB for care tracking
e. Step Functions for orchestration
f. EventBridge for workflow triggering
g. Pinpoint for notifications
